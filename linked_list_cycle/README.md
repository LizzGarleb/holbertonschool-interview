# Linked List Cycle
![Linked List Cycle](https://media.giphy.com/media/XK3lEP4lDwyObDPYFl/giphy.gif)

## Requirement
- Allowed editors: `vi`, `vim`, `emacs`
- Programs and function will be compiles with `gcc` 4.8.4` usign the flags `-Wall -Werror -Wextra -pedantic`
- All files should end with a new line
- No global variable allowed
- No more than 5 function per file
- Standard library allowed
- The prototypes of all functions should be included in your header file called `lists.h`

Write a function in C that checks if a singly linked list has a cycle in it.

- Prototype: `int check_cycle(listint_t *list);`
- Return: `0` if there is no cycle, `1` if there is a cycle

Requirements:
  - Only these functions are allowed: `write`, `printf`, `putchar`, `puts`, `malloc`, `free`
